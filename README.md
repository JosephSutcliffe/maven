# Maven
> /meɪvən/ – a trusted expert in a particular field, who seeks to pass timely and relevant knowledge on to others in the respective field.


## Install
```
pip install maven
```


## Usage
```python
import maven
maven.get('general_election-gb-2015-results', data_directory='../data/')
```
